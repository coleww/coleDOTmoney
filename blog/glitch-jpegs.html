Open a JPEG in a text editor and you’ll be treated to a wall of hex. Some file formats would not be happy if you started manipulating bits all willy nilly, but JPEGs are different. You can paste in selections from moby dick and something will still probably render when you open it as an image. The only thing you have to watch out for is the first few lines of data which contain important header information for the file. Just skip the first 1/10 of the file or so before making your edits.

swapping the order of lines messes it all up. changing N has differing effect. high N => data stays together but more out of place. lower N => data stays close to where it started but more shuffling.
